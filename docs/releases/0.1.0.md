# 0.1.0

*Initial release — 2026-03-03*

## Features

- **Core calendar component** (`CalendarWidget`) with month grid view, header navigation, and day cells
- **Date selection** — single date and multi-date selection via `value` / `onChange`
- **Date constraints** — `minDate`, `maxDate`, and `disabledDates` props to restrict selectable dates
- **Localization** — `locale` prop for `Intl.DateTimeFormat`-based month/year and weekday labels
- **Configurable week start** — `weekStartsOn` prop (0–6)
- **CSS custom properties** — 14 design tokens for colors, sizing, fonts, and border radius
- **Accessibility** — full keyboard navigation (arrow keys, Home/End, PageUp/PageDown), roving tabindex, ARIA roles (`grid`, `gridcell`, `columnheader`), `aria-selected`, `aria-disabled`, `aria-current="date"`, `aria-live="polite"` on month label
- **Public API exports** — `CalendarWidget`, `CalendarHeader`, `CalendarDayCell` components; `useCalendarState` hook; `getCalendarDays`, `isSameDay`, `isDateInRange`, `isDateDisabled`, `formatMonthYear` utilities; all TypeScript prop interfaces

## Build & Tooling

- Vite library build producing ES module (`.mjs`) and CommonJS (`.cjs`) bundles
- TypeScript declaration files in `dist/types/`
- CSS output in `dist/core.css`
- ESLint + Prettier configuration
- Vitest testing infrastructure with jsdom environment and `@testing-library/react`
