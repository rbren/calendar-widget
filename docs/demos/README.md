# Demos

## [Basic Calendar](./basic-calendar/)

A complete React app showcasing all CalendarWidget features:

- Single date selection
- Date range selection with hover preview
- Constrained range selection (min/max bounds)
- Min/max date range
- Disabled dates
- Localization (locale picker + week start day)
- Multi-date selection
- ISO week numbers
- Custom day rendering (emoji markers via `renderDay`)
- Lifecycle callbacks (`onMonthChange` / `onDayFocus` event log)
- Quick navigation disabled mode
- Dark theme
- Round cells theme
- Compact theme

### Running the demo

```bash
cd docs/demos/basic-calendar
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.
